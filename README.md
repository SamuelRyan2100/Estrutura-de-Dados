# Estrutura de Dados em C

Este repositório reúne implementações de **Estruturas de Dados e Algoritmos fundamentais** desenvolvidas em linguagem C para fins acadêmicos e de prática em programação.

O objetivo do projeto é estudar e aplicar conceitos importantes da área de **Estruturas de Dados**, implementando manualmente diferentes estruturas e algoritmos clássicos, com foco na manipulação de memória, ponteiros e análise de complexidade.

Os códigos são organizados em pastas de acordo com o tipo de estrutura ou algoritmo, permitindo uma melhor visualização e estudo de cada conceito.

## Conteúdo do Repositório

O projeto inclui implementações das seguintes estruturas e algoritmos:

### Estruturas Lineares

- Lista encadeada simples
- Lista duplamente encadeada
- Pilha
- Fila
- Fila de prioridade

### Algoritmos de Ordenação

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

Cada implementação busca demonstrar de forma prática:

- Alocação dinâmica de memória (`malloc`)
- Liberação de memória (`free`)
- Manipulação de ponteiros
- Implementação de algoritmos clássicos
- Organização e estruturação de código em C

## Objetivo

Este repositório foi criado como parte do processo de aprendizado na disciplina de **Estrutura de Dados**, servindo como um portfólio de implementações de algoritmos e estruturas fundamentais da computação.

Além do estudo acadêmico, o projeto também busca manter boas práticas de organização de código e documentação.

## Linguagem Utilizada
